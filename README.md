# cumulus

## Running

Run the following commands to serve locally on port `3000`:

```bash
$ npm install
$ npm run start
```

## Endpoints

### keywords

__GET__ `/api/keywords?q=<query>`

### companies

__GET__ `/api/companies?q=<query>`

### majors

__GET__ `/api/majors?q=<query>`
